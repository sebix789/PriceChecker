Price Checker

Price Checked is a web project created for fighting with scammed capitalism.

The main goal of the project is to show how different stores changes their products prices in large time period based on different holidays in the year.

How it works

Search for product you want to buy or look for and wait for the results.

If application not shows you results immediately, it means that the product is not in the database yet.

In case of that please be patient, because the application automatically run spider throught the web and search for you the best offers for the product you look for.


As you saw the result, the current product already be put to the database, so the app will be able to periodically check the current price. And then you will find the best time to buy it.

Every user is able to see different products by choosing the shop name from the dropdown list on the result subpage.


You could also see all products exist in database by choose "Show All" in the navigation bar.

There apart of flitering by shop, you could use the arrow buttons to sort by time, so you could see the exact price changing.

Please note that "Show All" subpage shows you all gained data, so if you not find there your product, click on the logo to go to the main page and gave the application ability to monitor product you look for, by searching it.
